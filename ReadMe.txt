These files accompany "Introduction to Embedded Systems using the MSPM0+" 
Jonathan W. Valvano, ISBN: 978-1466468863, copyright (c) 2026

Most of the folders contain CCS projects. Folders that begin with the name RTOS_ are ECE445M lab starter projects

These folders do not contain CCS projects
  The documentation folder contains html descriptions created by Doxygen
  The datasheets folder contains data sheets for components used in this curriculum
  The RTOS_Labs_common folder contains shared files for the RTOS labs  
  The inc folder contains general shared files. 
  The SensorBoard_v8 folder contains design files for the sensor board
  The MotorBoard_v7 folder contains design files for the motor board

TExaSdisplay is a Windows application that implements a serial terminal, oscilloscope and logic analyzer.
